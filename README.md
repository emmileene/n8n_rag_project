## Steps to Import a JSON file into n8n
1. Download the JSON file (PIMENTEL N8N RAG PROJECT.json)
2. Open your n8n dashboard
3. Go to Overview
4. Click "Create workflow"
5. Click the "..." three dots menu (top right)
6. Click "Import from File"
7. Select your JSON file from your computer
8. Click "Open"
9. Your workflow will load automatically
